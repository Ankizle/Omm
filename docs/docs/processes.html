Methods


Methods are blocks of code that are reusable and modifiable by parameters.

To declare a process in Omm you can use the proc keyword.

proc ~ myProc() {
-- do stuff
}

Return

Return statements are used to prevent the rest of the process from executing or send a value to a variable. The syntax for return statements is:

return ~ 'Your Value'; -- Your Value

Method parameters

Parameters are variables that only accessible within the process. Every time you call the process you can change the parameters.

proc ~ myMethod(param1, param2) {
-- do stuff
}

Calling a function

The syntax to call a function is kind of complicated. You will can use the #~ keyword to call a process.

proc ~ myMethod(param1, param2) {
-- do stuff
}

#~myProc('value 1', true)
