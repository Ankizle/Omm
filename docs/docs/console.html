Console


A key part of programming is console interaction. This is where we output what our code does. In Omm we can use the log keyword to output.

log ~ 'This Is My Output'

Also we have to take in data from the user. This is where reading from the console is helpful. This allows our user to input a piece of data, and we can store it in a variable. We can use the read keyword to read from the console. The read data is always stored as a string.

let ~ userString: read ~ 'Enter Your String';

It is important to note that the read function only reads when it is inside an expression.
