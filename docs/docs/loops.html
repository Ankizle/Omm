Loops


Loops are just conditions that execute while the statement is true. If you already know javascript, these loops are most comparable to while loops. It is important to note that a loop will go on infinitely unless the given condition is false.

let ~ myVar: 0;
loop (myVar < 5) {
-- do stuff
}

let ~ myVar: 0;
loop (myVar < 5) {
-- do stuff
myVar: 1 + myVar;
}

The first example makes an infinite loop, and the second example will stop after 5 iterations
