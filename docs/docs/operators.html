Operators


Operators are a key concept in mathematics. It is useless to even explain them because they are probably the first thing taught in school. Omm has 4 basic arithmetic operators:

  1. + (Addition)
  2. - (Subtraction)
  3. * (Multiplication)
  4. / (Division)
  5. % (Modulus)
Omm also has other operators, i.e exponentiation (^), string concatenation (+), connector (~), and assignment(:). We have already discussed assignment and the connector in this section and you already know exponentiation. If you are new eto programming then, you might not know string concatenation. This is just the process of joining two strings.

let ~ myVar: 'String 1' + 'String 2';

Omm follows a modified version of PEMDAS, it is PEMDMASSCCA. Yes, it is hard to remember but it is Parenthesis, Exponentiation, Multiplication / Division, Modulus, String Concatenation / Addition / Subtraction, Connector, Assignment. You may not know what modulus is if you do not know code already, but it just divides the numbers and gets the remainder.
