Conditionals


Conditionals are blocks of code that only execute if the given condition is true, e.g

if (1 = 1) {
--do stuff
}

If you want to execute something if a given statement is true, and if that statement is false it will execute something else you can use the if... else blocks.

if (1 = 1) {
--do stuff
} else {
--do other stuff
}

If you want to execute a block of code if something is true, execute another block if that is true, and execute another block if neither is true then use the elseif block.

if (1 = 1) {
--do stuff
} elseif (0 = 1) {
--do other stuff
} else {
--do other stuff
}

--you can use any amount of if, else if, and else blocks

Logic Gates

Logic gates are just compound conditions. Two or more conditions combined. There are 3 basic logic gates:
  1. AND gate
  2. OR gate
  3. NOT gate
Of course there are many more logic gates. There is an XOR, NAND, XNOR, etc. Many languages do not have support for shorthand complex logic gates. E.g, an XOR (Exclusive OR) is just an AND and an OR gate. Omm has support for these logic gates as shorthand.
  1. AND (&)
  2. OR (|)
  3. NOT (!)
  4. NAND (!&)
  5. NOR (!|)
  6. XOR ($|)
  7. XNOR ($!|)
