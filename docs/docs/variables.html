Variables


Variables are a key part of any programming language. In many languages variables are declared with what datatype it will be (int, double, String, boolean, float, char), most notably in Java and C++, but some languages use the var keyword, most notably in javascript. Omm uses the let keyword to declare variables.

let ~ myVariable: 100;

If you know any other programming language you will notice one prominent thing that is different in Omm than any other language, the tilde (~). This is called a connector statement in Omm. It is there because Omm removes all whitespace before reading the code. This connector "connects" the two tokens and makes the code readable by the lexer.

Datatypes

A key concept in programming is the datatype. This is just the type of data that is stored in a variable. Omm has 4 datatypes:
  1. Number - Any Number
  2. String - A piece of text
  3. Boolean - A value that is either true or false
  4. Array - A bunch of data
  5. Glossary - A bunch of unordered data that is labeled
  6. Falsey - An undefined or null value
To get the datatype of an expression, you can use the typeof keyword.

let ~ datatypeTest: typeof ~ 'test string'; --will return 'string'

Changing a variable

Sometimes in programming you need to change the variable's value. You can do that in Omm as well.

let ~ myVariable: 'This is my value';

MyVariable: 100; -- Changed Value

You can also re-use the let keyword to change the variable.

let ~ myVariable: 'This is my value';

let ~ MyVariable: 100; -- Changed Value

Global Variables

In programming there is an important factor called scope. The let keyword keeps the scope of the variable as a local variable. We will delve deeper into scope here, but just know that to declare a variable use the global keyword instead of let.

let ~ myLocalVar: 'Hello World';

global ~ myGlobalVar: 'Hello Globe';
